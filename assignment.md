## Assignment: Student Result Summary

- File to create: student-result.js
- Requirements: Store student name and score in variables.
- Grading Rules:
70 and above = A
60 to 69 = B
50 to 59 = C
40 to 49 = D
Below 40 = F

Pass Rules:
50 and above = Pass. Below 50 = Fail.

**Example Expected Output:**
`
Student Name: Amina
Score: 72
Grade: A
Status: Pass
`
