# Assignment: Student Result Summary

## Objective
Write a JavaScript program that stores a student's name and score, then displays the student's result summary based on the grading and pass rules below.

## File to Create
`student-result.js`

## Requirements
Your program must:

1. Store the **student name** in a variable.
2. Store the **student score** in a variable.
3. Determine the **grade** using the rules below.
4. Determine the **pass status** using the rules below.
5. Print the result in the expected format.

## Grading Rules

- **70 and above** = `A`
- **60 to 69** = `B`
- **50 to 59** = `C`
- **40 to 49** = `D`
- **Below 40** = `F`

## Pass Rules

- **50 and above** = `Pass`
- **Below 50** = `Fail`

## Example Expected Output

```text
Student Name: Amina
Score: 72
Grade: A
Status: Pass
